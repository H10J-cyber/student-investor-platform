# Student Investor & Economics Analytics Platform

Minimal scaffold: Streamlit app for finance and economics learning.

Quick start

1. Create a virtual environment and install dependencies:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2. Run the app:

```bash
streamlit run app.py
```

Notes

- `yfinance` is used for stock data.
- Economics data uses World Bank and REST Countries APIs.
- The app includes an Economic Heatmap & Country Analysis section for interactive global economic comparison.
